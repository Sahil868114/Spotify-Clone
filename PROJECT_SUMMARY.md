## 🚀 How to Use

### Quick Start (3 Steps)

1. **Add Your Songs**
   ```
   Create a 'songs' folder with:
   - song1.mp3
   - song2.mp3
   - song3.mp3
   - song4.mp3
   - song5.mp3
   ```

2. **Run Local Server**
   ```bash
   cd "c:\Users\Sahil saini\Spotify Clone"
   python -m http.server 8000
   ```

3. **Open in Browser**
   ```
   http://localhost:8000
   ```

### Keyboard Shortcuts
- **Space** - Play/Pause
- **→ Arrow** - Next Song
- **← Arrow** - Previous Song
