🔧 CUSTOMIZATION
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Add More Songs:
  1. Add MP3 files to 'songs/' folder
  2. Edit 'script.js' - Find 'let songs = [...]'
  3. Add new song object:
     {
         songName: "Title",
         filePath: "songs/songX.mp3",
         coverPath: "image_url",
         artist: "Artist"
     }

Change Colors:
  1. Edit 'style.css'
  2. Find ':root { ... }' at top
  3. Modify color values
  4. All colors update automatically

🐛 TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Audio not playing?
  → Check MP3 files are in 'songs/' folder
  → Use local server (not direct file access)
  → Check browser console for errors

Search not working?
  → Make sure song names match in script.js
  → Try searching by artist name
  → Clear browser cache

Favorites not saving?
  → Check if localStorage is enabled
  → Try private/incognito mode
  → Clear browser cache

Theme not switching?
  → Check console for errors
  → Try clearing cache
  → Verify CSS loads correctly
